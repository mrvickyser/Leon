## Leon
**A Simple WhatsApp Bot.**

## ⚙️ Setup ⚙️

```
pkg install git
pkg install nodejs
git clone https://github.com/TOXIC-DEVIL/Leon
cd Leon
npm i @adiwajshing/baileys
npm i chalk
node index.js
```
